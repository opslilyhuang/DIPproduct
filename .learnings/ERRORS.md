# 错误记录

## 格式
- **ID**: 唯一标识符
- **时间戳**: 错误发生的时间
- **严重程度**: 致命/严重/中等/轻微
- **错误描述**: 发生了什么问题
- **环境信息**: 系统、工具、版本等
- **根因分析**: 为什么会出现这个错误
- **解决方案**: 如何修复
- **预防措施**: 如何避免再次发生

## 记录

### 2026-03-07：Skill安装失败
- **ID**: ERR-001
- **时间戳**: 2026-03-07 19:25
- **严重程度**: 中等
- **错误描述**: 使用skillhub安装Ontology和Office-Automation时返回404错误
- **环境信息**: 
  - 系统: Linux
  - 工具: skillhub
  - 技能名称: ontology, office-automation
- **根因分析**: 
  1. 技能名称在skillhub注册表中不存在
  2. 技能名称可能有变化
  3. 技能可能已下架或移至其他源
- **解决方案**:
  1. 尝试其他安装方法（如npx clawhub@latest install）
  2. 搜索替代技能
  3. 手动下载技能包
- **预防措施**:
  1. 安装前先搜索确认技能名称
  2. 准备好备用安装方案
  3. 记录成功的安装方法
